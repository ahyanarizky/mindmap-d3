# mindmap-d3

## Install Global Package

```
npm install -g nodemon liveserver express-generator bower
```

## Run Server

```
npm install
npm start
```

## Run Client

```
bower install
live-server
```

## Endpoint

| Route       | HTTP          | Description  |
| :-------------|:-------------:| :-----:|
| http://localhost:3000/api/maps   | GET      | read all data |
